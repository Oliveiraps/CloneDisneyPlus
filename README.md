# Clone Disney Plus - Instruções de Desenvolvimento

## Como iniciar o projeto

Para abrir o projeto e iniciar o servidor de desenvolvimento diretamente no VS Code:

1. Abra o VS Code.
2. Abra um novo terminal integrado (Ctrl + ` ou View > Terminal).
3. Cole e execute o seguinte comando:

```bash
cd /home/izmael/Documentos/EZTUDOZ/EBAC/prjeto-04-cloneDisneyPlus/CloneDisneyPlus && npm run dev
```

Isso navegará para a pasta raiz do projeto e iniciará o Gulp em modo watch para desenvolvimento.

## Scripts disponíveis

- `npm run dev`: Inicia o desenvolvimento com monitoramento de arquivos.
- `npm run build`: Gera a versão de produção.

Certifique-se de ter as dependências instaladas com `npm install` antes de executar.
